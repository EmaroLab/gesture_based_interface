Detailed Description

Class to implement Background Segmentation.

The class removes the background, associated to the current Kinect angle, from the raw Point Cloud acquired by the Kinect and saved in /camera/depth/points. The filtered point cloud is published on the topic /camera/pcl_background_segmentation

Handler:

  • subscribe to /cur_tilt_angle to know the current tilt angle of the Kinect
  • publish filtered point cloud on /camera/pcl_background_segmentation
