## Requirements
* Linux Ubuntu 18.04
* Android Studio
* Ros Melodic
* ROSJava
* Smartwatch Android
* Estimote Beacons

## Package Content
Android Wearable application which utilizes ROSJava to publish presence messages on different topics, one for each beacon, whenever the wearer is inside the area of a beacon.
