## Requirements
* Linux Ubuntu 18.04
* Android Studio
* Ros Melodic
* ROSJava
* Smartwatch Android
* Estimote Beacons

## Package Content
Android Wearable application to detect if the wearer is in the area of the beacons.
This application utilizes ROSJava to publish presence messages on different topics, 
one for each beacon.
