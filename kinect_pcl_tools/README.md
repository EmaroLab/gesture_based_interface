## Package Content

This package contains the nodes to manage and filter the pointclouds acquired by the Kinect.

## Requirements
* Linux Ubuntu 18.04
* Ros Melodic
* Kinect 360 or equivalent

## Nodes
* pcl_record
* pcl_background_segmentation
* pcl_filter
